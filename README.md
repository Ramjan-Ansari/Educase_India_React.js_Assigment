# Educase India ReactJS Assignment

A responsive React.js application built for the Educase India assignment, implementing the PopX onboarding flow with user registration, login, and profile management screens.

---

## 🚀 Live Demo
Link: [https://your-deployed-link.vercel.app](https://educase-india-reactjs-assignment-z3.vercel.app/)

---

## 📌 Features

- Landing page with navigation
- User Signup form with validation
- Login functionality
- Profile page UI
- Reusable components (Button, InputField, Title)
- React Router navigation
- Dynamic button state handling
- Mobile-first fixed 375px layout design

---

## 🛠️ Tech Stack

- React.js
- Vite
- React Router DOM
- Tailwind CSS

## ⚙️ Installation & Setup

### 1. Clone repository
```bash
git clone https://github.com/Ramjan-Ansari/educase-india-reactjs-assignment.git
```

### 2. Go to project folder
```bash
cd frontend
```

### 3. Install dependencies
```bash
npm install
```

### 4. Run development server
```bash
npm run dev
```

---

## 📦 Build

```bash
npm run build
```
